What is a Library in Python?

A library, known in Python as a module, is an external component of code that you can import into your code. Essentially, it's a file full of functions which you can do cool things with. For example, a flightLog module may have a function that calls from a flight API and returns flight duration data for flights that you pass through the function. This removes the need to create your own function, get your own API key, or do all the mechanics yourself. The module has it all pre-built, and all you need to do is import the module and call some functions.

Tensorflow is an open-source library used for machine learning and artificial intelligence. It's used for the training of deep neural networks. It has much more precision than other modules such as Scikit, for example, it's graphing features are more complex. It has a large community, so help is easy obtainable. Scikit-learnSoftware Library

Scikit-learn is a machine learning library for Python. It's best used for classification, regression patterns, clustering, and other methods of analysis.

NumpySoftware Library

Numpy is a library used for performing complex mathematical operations on multiple different data types, such as multi-dimensional arrays and matrices.

PytorchModule

Pytorch is a machine learning framework. It's the largest ML library, which allows users to perform GPU-intensive computations, including rendering dynamic graphs and calculating gradients. It's used for applications such as computer vision and natural language processing.
